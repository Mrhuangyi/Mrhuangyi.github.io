L1-013 计算阶乘和（10 分）
对于给定的正整数N，需要你计算 S = 1! + 2! + 3! + ... + N!。

输入格式：

输入在一行中给出一个不超过10的正整数N。

输出格式：

在一行中输出S的值。

输入样例：

3
输出样例：

9

```cpp
#include<stdio.h>
int jie(int n){
    int i,t;
    t=1;
   for(i=1;i<=n;i++){
    t*=i;
}
return t;
}
int main()
{
   int i,n,sum;
   sum=0;
   scanf("%d",&n);
   for(i=1;i<=n;i++)
   {
       sum+=jie(i);
   }
   printf("%d",sum);
    return 0;
}

```
