L1-012 计算指数（5 分）
真的没骗你，这道才是简单题 —— 对任意给定的不超过10的正整数n，要求你输出2^n^。不难吧？

输入格式：

输入在一行中给出一个不超过10的正整数n。

输出格式：

在一行中按照格式“2\^n = 计算结果”输出2^n^的值。

输入样例：

5
输出样例：

2^5 = 32

```cpp
#include<stdio.h>
#include<math.h>
int main()
{
 int n,t;
 scanf("%d",&n);
 t=pow(2,n);
 printf("2^%d = %d",n,t);
    return 0;
}

```
