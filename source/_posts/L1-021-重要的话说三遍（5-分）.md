L1-021 重要的话说三遍（5 分）
这道超级简单的题目没有任何输入。

你只需要把这句很重要的话 —— “I'm gonna WIN!”——连续输出三遍就可以了。

注意每遍占一行，除了每行的回车不能有任何多余字符。

```cpp
#include<stdio.h>
#include<string.h>
int main()
{ int n;
  n=3;
  char a[]={"I'm gonna WIN!"};
  while(n--){
    printf("%s\n",a);
  }
 return 0;
}

```
