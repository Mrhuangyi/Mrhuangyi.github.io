L1-022 奇偶分家（10 分）
给定N个正整数，请统计奇数和偶数各有多少个？

输入格式：

输入第一行给出一个正整N（<= 1000）；第2行给出N个正整数，以空格分隔。

输出格式：

在一行中先后输出奇数的个数、偶数的个数。中间以1个空格分隔。

输入样例：

9
88 74 101 26 15 0 34 22 77
输出样例：

3 6

```cpp
#include<stdio.h>
#include<string.h>
int main()
{
    int n,m;
    int x,y;
    x=y=0;
 scanf("%d",&n);
 while(n--){
    scanf("%d",&m);
    if(m%2==1)
        x++;
    else
        y++;
 }
 printf("%d %d",x,y);
 return 0;
}
```
